**🎯 Project Overview**
Developed an AI-powered intelligent document processing workflow that extracts structured information from unstructured PDF documents using Langflow, Groq LLM, and Prompt Engineering.

**⚙️ Workflow Architecture**

**📖 Read File Node**
Upload PDF documents
Extract raw textual content

**📝 Prompt Template Node**
Inject extracted text into structured prompt
Standardize extraction instructions

**🤖 Groq LLM Node**
Process raw text using Groq-hosted Llama 3.1
Extract target structured fields intelligently

**💬 Chat Output Node**
Display extracted data in readable structured format

**🛠️ Tech Stack**

**Category	Tools / Technologies**
